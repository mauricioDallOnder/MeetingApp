A meeting app built using React and Chakra UI, with VideoSDK.live SDK integration for video conferencing.

## Getting Started

First, follow the steps below:

```bash
Clone this repository: gh repo clone mauricioDallOnder/MeetingApp
# 
Navigate to the project folder
# 
Install the dependencies: npm install
#
Start the app in development mode: npm run dev

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


## Features

- Creating and entering meeting rooms.
- Video and audio transmission,screen sharing.

## Settings

To use the API and SDK, you need to configure the apiUrl, getTokenUrl and deactivateRoomUrl in the MeetingContainer.tsx file. Be sure to replace these URLs with the corresponding URLs provided by your service provider.
